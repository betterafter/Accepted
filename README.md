# Accepted

## 소개

상자를 옳바른 위치에 모두 넣는, 창고지기라고도 불리는 소코반(Sokoban) 형식의 2D 퍼즐 게임입니다.

## 사용 기술 및 응용 SW

- Unity3D
- C#
- Google Play Games Service
